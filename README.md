# TANGO-CHO（単語帳 × DeepL × 類義語 × 4択クイズ + Fortune(EN)）PWA〔TOEIC寄り〕

英語を読んでいて分からない単語が出たら、**見つける → 取り込む → 翻訳 → 保存**。  
集めた単語を **単語カード** と **4択クイズ** で回して定着させる、軽量PWAです。

- **Live Demo**：https://masato-nasu.github.io/TANGO-CHO/

## できること / Features

- 📥 **単語の取り込み**
  - コピーした単語を貼り付けて追加
  - URLパラメータ `?word=` による受け取り（ショートカット等で活用）
- 🌐 **DeepL翻訳**（意味の取得）
- 🧠 **類義語の提示**（語彙の広がり）
- 🃏 **単語カード**（意味・例文・発音・品詞をまとめて確認）
- ✅ **4択クイズ**（記憶の定着）
- 🔊 **発音**（端末の音声読み上げ機能を利用）
- 🧭 **Fortune(EN)**（英文を読む “裏導線” としての学習導線）
- 🏷️ **品詞表示（名／動／形／副）**
  - アプリ内に同梱された辞書データにより、**追加単語でも自動で品詞候補を表示**
  - ※品詞は「断定」ではなく候補表示（誤学習を避ける方針）
- 💾 **JSONエクスポート／インポート**（バックアップ・共有）
  - iPhone/Androidどちらでも利用可能（方法は後述）

## 使い方 / How to use

### 1) 単語を追加する
1. 追加したい英単語を入力（または貼り付け）
2. 翻訳（DeepL）
3. 保存

> 追加後は「単語カード」で確認し、「クイズ」で反復するのがおすすめです。

### 2) クイズで回す
- 苦手単語だけ回す／ランダムで回す、などの導線で反復できます。

### 3) JSONでバックアップする
- データは端末内（ブラウザ内）に保存されます。  
  端末変更や不測の削除に備え、定期的にJSONエクスポートを推奨します。

## インストール（PWAとしてホーム画面に追加）

### iPhone（iOS）でのインストール手順
> **必ずSafariで**開いてください（LINE内ブラウザ/一部ブラウザだと「追加」が出ない場合があります）。

1. iPhoneで **Safari** を開く  
2. https://masato-nasu.github.io/TANGO-CHO/ を開く  
3. 画面下（または上）の **共有ボタン（□に↑）** をタップ  
4. **「ホーム画面に追加」** をタップ  
5. 名前を確認して **追加**

✅ ホーム画面のアイコンから起動すると、アドレスバーがない “アプリ表示” になります。

#### iPhoneで更新が反映されない場合（重要）
PWAはキャッシュが強いです。次の順でお試しください。

1) PWAを完全終了（アプリスイッチャーで上に払って終了）→ 再起動  
2) それでもダメなら：  
   **設定 → Safari → 詳細 → Webサイトデータ** で `masato-nasu.github.io` を削除 → 再度Safariで開く → もう一度「ホーム画面に追加」  
3) 最終手段：ホーム画面のTANGO-CHOアイコンを削除 → 再インストール

---

### Androidでのインストール手順
> 推奨：**Chrome**（一部ブラウザでも可能ですが表示が異なる場合があります）

1. Androidで **Chrome** を開く  
2. https://masato-nasu.github.io/TANGO-CHO/ を開く  
3. 右上の **︙（メニュー）** をタップ  
4. **「アプリをインストール」** もしくは **「ホーム画面に追加」** を選択  
5. 表示に従ってインストール

✅ ホーム画面にアイコンが作成され、アプリのように起動できます。

#### Androidで更新が反映されない場合
- Chromeで該当ページを開き、再読み込み  
- それでもダメなら（強め）：  
  Androidの設定から Chrome のサイトデータを削除、またはアプリ（PWA）を一度アンインストール→再インストール

## JSONインポート / エクスポート（バックアップ・共有）

### エクスポート（保存）
- アプリ内のエクスポート機能からJSONを出力し、端末へ保存します。

### インポート（復元）
- 保存したJSONを、アプリ内のインポート機能で読み込みます。

---

## データとプライバシー / Privacy
- 単語帳データは基本的に端末内（ブラウザストレージ）に保存されます。
- 翻訳のために、入力された単語がDeepLへ送信されます（翻訳取得時）。
